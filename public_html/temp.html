UX / UI Design Concept

Stucky's Stickers

Stucky's Stickers is a conceptual UX UI project focused on researching the use case for a local sticker production company with New Mexican's in mind. It encompassed UX UI principals in consumer research and LEAN production processes, brand identity development, low fidelity prototyping using Marvel, and high fidelity prototyping using Figma.

Collaborator: Luke Thresher

Technologies Used: Adobe Illustrator, Adobe Lightroom, Google Business Suite, Roland Versaworks for sticker production, Marvel, and Figma.
